# Contributing to 9PAY PHP SDK

Thank you for considering contributing to the 9PAY PHP SDK!

## Code of Conduct

Please be respectful and professional in all interactions.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Write tests for your changes.
4. Ensure all tests pass.
5. Submit a pull request.

## Coding Standards

This project follows [PSR-12](https://www.php-fig.org/psr/psr-12/) coding standards.

## Security

If you discover a security vulnerability, please send an e-mail to geckotech1001@gmail.com instead of using the issue tracker.
